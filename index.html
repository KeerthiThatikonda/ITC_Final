Addendum

createGrid():
The createGrid function initializes the game grid by first clearing the content of the HTML element with the ID 'gridContainer'. It then generates a 5x5 grid of div elements, each representing a cell. Event listeners are added to each cell, linking them to the toggleLights function. The grid is filled with random initial values (true or false), and the visual appearance of each cell is updated accordingly.

toggleLights(row, col):
The toggleLights function is responsible for toggling the state of the clicked cell and its adjacent cells in the up, down, left, and right directions. The function checks whether the game has started and exits if not. It then toggles the cells' states, increments the moves counter, updates the moves display, and checks for a win condition after each move.

toggleCell(row, col):
The toggleCell function toggles the state of a specific cell identified by its row and column indices if it is within the boundaries of the grid. It updates the cell's state in the grid and calls the updateCell function to reflect the change visually.

updateCell(row, col):
The updateCell function visually updates the background color of a specific cell based on its state in the grid. If the state is true, the background color is set to '#333' (dark); otherwise, it is set to '#eee' (light).

updateMoves():
The updateMoves function updates the displayed number of moves on the HTML element with the ID 'moves' by setting its text content to 'Moves: ' concatenated with the current number of moves.

startTimer():
The startTimer function initiates a timer using the setInterval method. It increments the seconds counter every second and updates the displayed timer with the formatted time.

stopTimer():
The stopTimer function clears the interval created by startTimer, effectively stopping the timer.

formatTime(seconds):
The formatTime function takes the elapsed time in seconds and formats it into a 'mm:ss' string, handling zero-padding for minutes and seconds.

checkWin():
The checkWin function checks if all cells in the grid are turned off (false). If so, it stops the timer, displays a congratulatory alert with the number of moves and time taken, and resets the game using the resetGame function.

startGame():
The startGame function resets the move and time counters, updates the moves display, creates a new grid, starts the timer, and sets the gameStarted flag to true.

resetGame():
The resetGame function stops the timer, creates a new grid, resets the timer and move counters, updates the moves display, and sets the gameStarted flag to false.

Initial setup:
The code concludes with the initial setup, calling the createGrid function to set up the initial game grid.
